# Shapes OpenGL
OpenGL shapes library for rendering 2D and 3D geometry (spheres, tori, cones, cylinders, arbitrary polynomials, etc.)

Building:
```
git submodule update --init --recursive
mkdir build && cd build
cmake -DBUILD_DEMOS=ON ..
cmake --build .
```
